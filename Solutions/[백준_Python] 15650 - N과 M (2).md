## 문제

자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.

- 1부터 N까지 자연수 중에서 중복 없이 M개를 고른 수열
- 고른 수열은 오름차순이어야 한다.

---

## 입력

첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)

---

## 출력

한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

---

## 해설

```python
n, m = map(int, input().split())


def find(cur_nums, cur_idx):
    global n, m
    if cur_idx == m:
        print(*cur_nums)
    else:
        for i in range(1, n + 1):
            if not cur_nums or i > max(cur_nums):
                cur_nums.append(i)
                find(cur_nums, cur_idx + 1)
                cur_nums.pop()


find([], 0)
```

1부터 N까지의 자연수 중 중복 없이 M개를 골라 오름차순으로 수열을 만들 때 가능한 경우의 수를 구하는 문제입니다.

재귀(Recursion)를 사용하여 하나씩 고르면서 수열을 만들도록 함수를 구현했습니다. 현재 몇 개를 골랐는지 나타내는 cur_idx와 현재 고른 숫자가 담겨있는 cur_nums를 사용하여 차례대로 구현했습니다. 이 때 현재까지 고른 숫자보다 더 큰 숫자를 고르도록(Line 10) 조건을 걸어 오름차순으로 고를 수 있도록 하였고 처음 함수에 호출할 때는 cur_nums가 비어있는 상태이기 때문에 해당 상태에서는 바로 숫자를 고르도록 구현했습니다.

보통 탐색의 경우 간단히 반복문을 사용하여 구현하는 경우가 많습니다. 그러나 깊이의 단계가 변수로 주어진 경우에 이를 재귀를 사용하여 구현할 줄 아는 스킬이 필요하며 이를 물어보는 문제였다고 생각합니다.
