## 문제

2×n 직사각형을 1×2, 2×1과 2×2 타일로 채우는 방법의 수를 구하는 프로그램을 작성하시오.

아래 그림은 2×17 직사각형을 채운 한가지 예이다.

![img](source/t2n2122.gif)

---

## 입력

첫째 줄에 n이 주어진다. (1 ≤ n ≤ 1,000)

---

## 출력

첫째 줄에 2×n 크기의 직사각형을 채우는 방법의 수를 10,007로 나눈 나머지를 출력한다.

---

## 해설

```python
n = int(input())
cache = [0 for _ in range(n)]

cache[0] = 1
if n > 0:
    cache[1] = 3
for i in range(2, n):
    cache[i] = cache[i - 1] + 2 * cache[i - 2]

print(cache[-1] % 10_007)
```

2 x n 크기의 직사각형을 주어진 세 가지 타일로 채우는 방법의 수를 구하는 문제입니다.

먼저 가로 크기가 1인 타일은 한 종류, 가로 크기가 2인 타일은 두 종류가 존재합니다. 즉 2 x i 크기의 직사각형이 있다면 해당 직사각형을 주어진 타일로 채우는 방법의 수는 2 x (i-1) 크기의 직사각형을 타일로 채우는 방법의 수에  2 x 1 타일을 하나 붙이는 경우(1)와 2 x (i-2) 크기의 직사각형을 타일로 채우는 방법의 수에 2 x 1 타일을 두 개 붙이는 경우(2)와 2 x 2 타일을 하나 붙이는 경우(3)으로 나눌 수 있습니다.

즉, 2 x i 크기의 직사각형을 주어진 타일로 붙이는 방법의 수는 2 x (i-1) 크기의 직사각형 방법의 수와 2 x (i-2) 크기의 직사각형 방법의 수 * 2의 합임을 알 수 있습니다.
$$
solution(i) = solution(i-1) +2 \times solution(i-2)
$$

위 규칙을 기반으로 동적계획법(DP)을 사용하여 상향식으로 경우의 수를 저장했고 마지막에 10,007로 나눈 나머지를 출력함으로써 문제 풀이에 성공했습니다.
