## 문제

N개의 자연수와 자연수 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오. N개의 자연수는 모두 다른 수이다.

- N개의 자연수 중에서 M개를 고른 수열

---

## 입력

첫째 줄에 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)

둘째 줄에 N개의 수가 주어진다. 입력으로 주어지는 수는 10,000보다 작거나 같은 자연수이다.

---

## 출력

한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

---

## 해설

```python
n, m = map(int, input().split())
nums = sorted(list(map(int, input().split())))


def find(cur_nums, cur_idx):
    global n, m
    if cur_idx == m:
        print(*cur_nums)
    else:
        for num in nums:
            if num not in cur_nums:
                cur_nums.append(num)
                find(cur_nums, cur_idx + 1)
                cur_nums.pop()


find([], 0)
```

N개의 자연수 중 M개를 골라 가능한 수열들을 출력하는 문제입니다.

먼저 수열은 사전 순으로 증가하는 순서로 출력되어야 하기 때문에 숫자를 뽑을 때 사전 순으로 증가하는 순서로 뽑히도록 하기 위해 N개의 숫자들을 저장하는 nums 리스트를 오름차순으로 정렬합니다.

그리고 재귀(Recursion)을 사용하여 숫자를 nums에서 하나씩 고르면서 수열을 만들 수 있도록 합니다. 이 때 수열을 이루는 숫자들은 모두 다른 수이기 때문에 수열에 있는 숫자는 뽑지 않도록(Line 11) 구현합니다. 그 외에는 재귀를 사용한 탐색 문제로 어렵지 않게 구현이 가능하다 생각됩니다.

현재 선택된 숫자들을 저장하는 cur_nums에서 find를 호출하고 나서 14번 라인에서 pop을 하는 이유는 참조로 cur_nums를 인자로 넘겼기 때문에 현재 cur_nums에는 선택된 숫자가 들어가 있는 상태입니다. 그러나 다른 경우의 수를 구할 때는 해당 경우의 수에서 선택한 숫자는 빠져야 하므로 pop을 사용하여 선택한 숫자를 빼도록 합니다.
