## 문제

상근이는 요즘 설탕공장에서 설탕을 배달하고 있다. 상근이는 지금 사탕가게에 설탕을 정확하게 N킬로그램을 배달해야 한다. 설탕공장에서 만드는 설탕은 봉지에 담겨져 있다. 봉지는 3킬로그램 봉지와 5킬로그램 봉지가 있다.

상근이는 귀찮기 때문에, 최대한 적은 봉지를 들고 가려고 한다. 예를 들어, 18킬로그램 설탕을 배달해야 할 때, 3킬로그램 봉지 6개를 가져가도 되지만, 5킬로그램 3개와 3킬로그램 1개를 배달하면, 더 적은 개수의 봉지를 배달할 수 있다.

상근이가 설탕을 정확하게 N킬로그램 배달해야 할 때, 봉지 몇 개를 가져가면 되는지 그 수를 구하는 프로그램을 작성하시오.

---

## 입력

첫째 줄에 N이 주어진다. (3 ≤ N ≤ 5000)

---

## 출력

상근이가 배달하는 봉지의 최소 개수를 출력한다. 만약, 정확하게 N킬로그램을 만들 수 없다면 -1을 출력한다.

---

## 풀이

```python
from math import inf

n = int(input())
dp = {i: inf for i in range(1, n + 1)}
dp[3] = 1
dp[5] = 1
for i in range(6, n + 1):
    dp[i] = min(dp[i - 3], dp[i - 5]) + 1

if dp[n] != inf:
    print(dp[n])
else:
    print(-1)

```

3킬로그램짜리 봉지와 5킬로그램짜리 봉지만을 사용하여 정확하게 N 킬로그램의 설탕을 담을 때 봉지의 최소 개수를 구하는 문제입니다.

가장 먼저 고려해야할 점은 정확히 N 킬로그램이 3킬로그램 봉지와 5킬로그램 봉지로 담을 수 없으면 -1을 출력해야 합니다. 때문에 3킬로그램의 설탕과 5킬로그램의 설탕을 각각 3킬로그램 봉지, 5킬로그램 봉지 하나로 담는 경우의 수부터 출발하여 6킬로그램이상부터(1, 2, 4킬로그램은 무조건 담을 수 없고, 3, 5킬로그램은 봉지 하나부터 시작합니다) 봉지의 최소 개수를 구합니다. 만약 8킬로그램을 예로 들면, 5킬로그램을 담는 봉지의 최소 개수에 3킬로그램 봉지를 하나 담으면 되기 때문에 추가됩니다. 여기서 코드로 구현시 주의할 점은 2킬로그램을 담는 봉지의 최소 개수에 5킬로그램 봉지를 하나 담은 경우의 수까지 고려하기 때문에 이 값이 저장되지 않도록 주의해야합니다. 저는 python 기본 패키지인 math의 inf 클래스를 사용하여 무한 값으로 예외처리하여 구현했습니다.

위 과정은 입력 받은 n까지 반복문을 통해 봉지의 최소 개수를 구한뒤 정확하게 N킬로그램을 만들 수 없으면, 즉 dp[n]이 inf이면 -1을 출력하고 아니면 해당 값을 출력하도록 하여 문제 풀이에 성공했습니다.
