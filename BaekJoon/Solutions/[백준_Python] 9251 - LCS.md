## 문제

LCS(Longest Common Subsequence, 최장 공통 부분 수열)문제는 두 수열이 주어졌을 때, 모두의 부분 수열이 되는 수열 중 가장 긴 것을 찾는 문제이다.

예를 들어, ACAYKP와 CAPCAK의 LCS는 ACAK가 된다.

---

## 입력

첫째 줄과 둘째 줄에 두 문자열이 주어진다. 문자열은 알파벳 대문자로만 이루어져 있으며, 최대 1000글자로 이루어져 있다.

---

## 출력

첫째 줄에 입력으로 주어진 두 문자열의 LCS의 길이를 출력한다.

---

## 풀이

```python
string_a = input()
string_b = input()

lcs = [[0 for _ in range(len(string_b) + 1)] for _ in range(len(string_a) + 1)]

for i, i_chr in enumerate(string_a, 1):
    for j, j_chr in enumerate(string_b, 1):
        if i_chr == j_chr:
            lcs[i][j] = lcs[i - 1][j - 1] + 1
        else:
            lcs[i][j] = max(lcs[i - 1][j], lcs[i][j - 1])

print(lcs[-1][-1])

```

문제 그대로 최장 공통 부분 수열의 길이을 찾는 문제입니다. 최장 공통 부분 수열이란 두 수열(문자열)이 주어졌을 때, 부분 수열이 되는 수열 중 가장 긴 수열을 의미합니다.

예를 들면 ACAYKP와 CAPCAK의 최장 공통 부분 수열은 ACAK로 길이는 4가 됩니다. 이러한 최장 공통 부분 수열(LCS) 문제의 경우 동적 계획법(DP)를 사용하여 풀이가 가능합니다. 각 문자열의 처음부터 시작하여 최장 공통 부분 수열을 저장해나가는 방식을 통해 각 문자열의 길이를 1부터 시작하는 부분 문제로 분할하여 해결이 가능합니다.

최장 공봉 부분 수열의 길이를 저장하는 2차원 배열을 lcs라 할때, 첫번째 문자열의 문자를 행, 두번째 문자열의 문자를 열로 나타낸다고 할 때 해당 인덱스에서 두 문자가 같으면 이전 행과 이전 열의 lcs 값에 1을 더한 값을, 두 문자가 다르면 이전 행 혹은 이전 열의 값 중 큰 값을 취하면 최장 공통 부분 수열의 길이를 구할 수 있게 됩니다. 이러한 과정을 표를 사용하여 나타내면 다음과 같습니다.

​	 0	A	C	A	Y	K	
0	0	0	0	0	0	0
C	0	0	1	1	1	1
A	0	1	1	2	2	2
P	0	1	1	2	2	2
C	0	1	2	2	2	2
A	0	2	2	3	3	3
K	0	2	2	3	3	4

이러한 알고리즘을 구현하기 위해 lcs 변수는 각 문자열의 길이+1의 크기를 가진 2차원 배열로 선언하고 0으로 초기화 하였습니다. 그리고 인덱스 1부터 시작하여 위에서 설명한 알고리즘을 그대로 구현하여 2차원 배열의 마지막 인덱스 값을 출력하면 두 문자열의 최장 공통 부분 수열의 길이를 출력하게 됩니다.
