## 문제

자연수 N과 M이 주어졌을 때, 아래 조건을 만족하는 길이가 M인 수열을 모두 구하는 프로그램을 작성하시오.

- 1부터 N까지 자연수 중에서 M개를 고른 수열
- 같은 수를 여러 번 골라도 된다.
- 고른 수열은 비내림차순이어야 한다.
  - 길이가 K인 수열 A가 A1 ≤ A2 ≤ ... ≤ AK-1 ≤ AK를 만족하면, 비내림차순이라고 한다.

---

## 입력

첫째 줄에 자연수 N과 M이 주어진다. (1 ≤ M ≤ N ≤ 8)

---

## 출력

한 줄에 하나씩 문제의 조건을 만족하는 수열을 출력한다. 중복되는 수열을 여러 번 출력하면 안되며, 각 수열은 공백으로 구분해서 출력해야 한다.

수열은 사전 순으로 증가하는 순서로 출력해야 한다.

---

## 풀이

```python
n, m = map(int, input().split())


def choose(idx, goal, nums):
    global n
    if idx == goal:
        print(*nums)
    elif idx < goal:
        start = 1 if not nums else nums[-1]
        for i in range(start, n + 1):
            nums.append(i)
            choose(idx + 1, goal, nums)
            nums.pop()


choose(0, m, [])
```

두 자연수 N과 M이 주어질 때, 1부터 N까지의 자연수 중 M개를 중복이 가능하게 선택하여 만든 오름차순인 순열을 출력하는 문제입니다.

M개의 자연수를 선택하는 과정에서, 첫번째 선택한 후 재귀를 통해 두번째 선택으로 넘어가고 이를 M번째 선택하는 과정까지 재귀를 통해 반복하는 과정을 통해 문제를 해결할 수 있습니다.

먼저 자연수 N과 M을 입력으로 받습니다. 그리고 총 m개의 선택 중 몇번째 선택인지 나타내는 idx와 현재 선택한 숫자를 담아놓는 nums 배열과 총 몇 번 선택해야하는지를 나타내는 goal를 사용하여 현재 idx번째 선택에서 선택한 숫자들이 nums에 담겨있을 때 가능한 경우의 수를 재귀를 통해 돌려보면서 idx와 goal이 같은 순간(M개의 선택을 마친순간) nums에 담겨있는 숫자들이 출력되도록 합니다.

그리고 선택된 수열은 오름차순이여야 하기 때문에 현재 선택된 숫자가 담겨있는 nums의 마지막 원소보다 크거나 같은 숫자들을 순회하여 nums의 원소들이 오름차순이 되도록 합니다.
