## 문제

45656이란 수를 보자.

이 수는 인접한 모든 자리수의 차이가 1이 난다. 이런 수를 계단 수라고 한다.

세준이는 수의 길이가 N인 계단 수가 몇 개 있는지 궁금해졌다.

N이 주어질 때, 길이가 N인 계단 수가 총 몇 개 있는지 구하는 프로그램을 작성하시오. (0으로 시작하는 수는 없다.)

---

## 입력

첫째 줄에 N이 주어진다. N은 1보다 크거나 같고, 100보다 작거나 같은 자연수이다.

---

## 출력

첫째 줄에 정답을 1,000,000,000으로 나눈 나머지를 출력한다.

---

## 풀이

```python
n = int(input())

count = [[-1 for _ in range(10)] for _ in range(n)]
count[0] = [1 for _ in range(10)]

for i in range(1, n):
    for j in range(10):
        if j == 0:
            count[i][0] = count[i - 1][1]
        elif j == 9:
            count[i][9] = count[i - 1][8]
        else:
            count[i][j] = count[i - 1][j - 1] + count[i - 1][j + 1]

print(sum(count[n - 1][1:]) % 1_000_000_000)

```

동적 계획법(DP)을 사용한 문제입니다. 처음에 재귀 호출을 사용하여 DP를 구현하려 했으나 반복문으로 간단히 나타낼 수 있을 것 같아서 반복문을 사용했습니다.

맨 앞자리의 수부터 시작해서 위 아래로 +-1을 갖는 다음 자리 수를 이진 트리로 생각해서 가장 끝자리의 경우의 수 1부터 시작하여 점점 더해가는 알고리즘으로 구현했습니다.

맨 마지막 자리의 경우의 수는 무조건 1이기 때문에 1로 초기화한후 두번째 자리서부터 현재 자리 수 +-1의 이전 자리 경우의 수를 더해가며 맨 앞자리의 모든 수에 대해 경우의 수를 구하고, 첫 자리의 수는 0이 될 수 없기 때문에 0을 제외한 경우의 수를 합한 후 조건대로 1,000,000,000으로 나눠 정답을 구했습니다.
